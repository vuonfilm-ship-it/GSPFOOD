# GSPFood Script Library — Deploy to Vercel

## Deploy in 3 steps (~60 seconds)

### 1. Push to GitHub
- Go to github.com → New repository → name it `gspfood-app`
- Upload all these files (drag & drop works)

### 2. Import to Vercel
- Go to vercel.com → Add New Project
- Import your `gspfood-app` GitHub repo
- Framework: **Vite** (auto-detected)
- Click **Deploy**

### 3. Share the URL
- Vercel gives you a live URL like `gspfood-app.vercel.app`
- Share with your team — works on any device, no login needed

## Local development (optional)
```
npm install
npm run dev
```
